# GO SYNTAX

# What is a syntax?

Syntax refers to rules and principles governing the arrangement of words, phrases and clauses to create well-formed programmes.

# WHAT ARE GO SYNTAX?

Go syntax is designed for simplicity, readability, and efficiency, including a clean C-like structure with mandatory package declarations, imported packages, and curly braces for code blocks.

The synytax consists of 
PACKAGE DECLARATION
IMPORT PACKAGES
FUNCTIONS
STATEMENTS AND EXPRESSION e.g

package main // package declaration
import ("fmt") // import packages

func main(){ // function declaration
    fmt.Println("Hello World") //statement
}
