# GO COMMENTS
 # What are go comments

 Go comments are text that are ignored upon execution.

 Comments are used in explaining codes, and to make it more readable.

 They are also used in preventing code execution when testing an alternative code.

 # THEY ARE 2 TYPES OF GO COMMENTS

 Go Single-line Comments:

 Single-line comments start with two forward slashes(//).

 Any text in between the slash // and at the end of the line is ignored during execution.

 ## Go Multi-line Comments
 Multi-line comments starts with /* and ends with */.
 So any text in between /* and */ will be ignored during execution.
