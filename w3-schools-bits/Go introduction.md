  # INTRODUCTION TO GO
  #   WHAT IS GO?

  Go often refer to as golang is an open source , statically typed, and compiled programming language developed by google.


 It was designed in 2007 by Robert Griesemer, Rob Pike,and Ken Thompson.

  # WHAT IS GO USED FOR?

  Go is used for website development mostly server-side.

  Go is also used in cloud-native development which is an approach to designing, building,and running applications that fulley leverage cloud computing models.

  # WHAT IS THE IMPORTANCE OF USING GO?
